Commands

.kill - Deletes all channels, recreates spam channels, and spams the server. .banall - Bans all members in the server. .kickall - Kicks all members in the server. .lock-c - Toggles lock on all channels. .nsfw-lock - Toggles NSFW lock on all channels. .timeout-everyone - Timeouts all members. .max-channels - Creates the maximum number of channels allowed. .admin-all - Gives everyone role all permissions and creates backup admin roles. .help - Displays this help menu.
